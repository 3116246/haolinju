  • 获取消息代理Token
    • 接口名称 getMessageProxyToken
    功能描述 特殊接口。该接口获取的token暂时仅能用于应用系统中通过后台自动推送消息时使用。该接口只支持POST方式提交的Http请求。
    该token不需要的用户认证即可获取,有效期为30秒，获取时的采用动态授权码，其动态授权码code生成规则及步骤为：
    1、使用appid和appkey顺序组合成一个新字符串A。例：appid为123456，appkey为89!{H45U,则组合后的A应该为：12345689!{H45U。
    2、对A进行MD5加密(32位)处理，得到新的密文C。例：C=MD5("12345689!{H45U")。
    3、将密文C做为授权码，在获取token时传入。
    接口地址 {Appcenter-HOST}/appcenter/oauth2/proxytoken
    参数列表
    参数名参数值说明
    appid 应用注册时由Wefafa分配的唯一标识
    openid Wefafa用户的唯一标识
    code 动态授权码
    grant_typeproxyproxy。
    state 应用状态码。由应用系统指定。Wefafa返回时会原样返回，应用系统可根据该值对本次请求结果进行校验
    返回值列表
    属性名属性值说明
    access_token API访问token码
    token_typebearer固定返回值
    expires 本次获取token的有效截止日期
    expires_in 本次获取token的有效时长。固定为30秒
    state 应用状态码。请求授权码时提交的应用状态码。
    error 错误代码。发生错误时返回
    msg 错误信息。发生错误时返回
