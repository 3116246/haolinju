    公司简介


  •         北京发发时代信息技术有限公司(Justsy Inc)成立于2011年10月，是一家专注于帮助企业客户提升信息技术管理与协作水平，通过其自主研发的Wefafa平台为企业客户提供全面的信息技术产品和服务的高科技互联网公司。

  •         发发时代公司致力于打造一个可全球协作的企业协作网络(Wefafa.com），让所有企业在协作网络平台上既可以内部员工间协同工作，通过信息技术提升企业管理水平、降低企业管理成本；又可以与合作伙伴及客户间网络化协作，通过互联网及移动互联网技术提升企业间协作水平，宣传企业产品与形象，并发现更多的商业机会。

  •         发发时代公司倡导开放、融合、互利、共赢的互联网精神，其核心平台Wefafa正是基于这种精神而诞生的，目的是让更多的中小企业能够平等的享有先进的信息技术及互联网技术给企业带来的价值，同时给包括大中型企业IT部门在内的第三方软件与服务企业创造更多的商业机会。

  •         发发核心团队均来自于国内外知名企业的CIO、产品总监、技术专家等，普遍有着10多年的企业级产品研发管理经验，深刻了解企业客户需求，我们拒绝平庸，不断追求技术创新。

  •         我们秉承“创新、协作、分享、共赢”的核心价值观，倡导人与人、企业与企业间随时随地沟通、协作、分享，利用协作网络的力量，与合作伙伴及客户一起实现互利共赢，共同为企业创造价值，为社会创造财富。
